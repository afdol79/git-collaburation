```
---------
```

# 2 การจัดรูปแบบข้อความ

```
2.1 ตัวหนา (Bold)
**ข้อความตัวหนา**
__ข้อความตัวหนา__
```

```
2.2 ตัวเอียง (Italic)
*ข้อความตัวเอียง*
_ข้อความตัวเอียง_
```

```
2.3 ตัวหนาและเอียง
***ข้อความตัวหนาและเอียง***
___ข้อความตัวหนาและเอียง___
**_ข้อความตัวหนาและเอียง_**
```

```
2.4 ข้อความขีดฆ่า (Strikethrough)
~~ข้อความที่ถูกขีดฆ่า~~
```

```
2.5 ข้อความเน้น (Highlight)
==ข้อความที่เน้น==
```

# 4 รายการ (Lists)

```
4.1 รายการไม่มีลำดับ (Unordered Lists)
- รายการที่ 2
  - รายการย่อย 2.1
  - รายการย่อย 2.2
- รายการที่ 3

* รายการที่ 1
* รายการที่ 2

+ รายการที่ 1
+ รายการที่ 2
```

```
4.2 รายการมีลำดับ (Ordered Lists)
1. รายการที่ 1
2. รายการที่ 2
    1. รายการย่อย 2.1
   2. รายการย่อย 2.2
3. รายการที่ 3
```

```
4.3 รายการกิจกรรม (Task Lists)
- [x] งานที่เสร็จแล้ว
- [ ] งานที่ยังไม่เสร็จ
- [x] งานที่เสร็จแล้ว
  - [ ] งานย่อยที่ยังไม่เสร็จ
  - [x] งานย่อยที่เสร็จแล้ว
```
